GWT(Google Web Toolkit),是google公司推出的一款基于JAVA的WEB应用开发工具包。通过GWT可以迅速实现基于RPC的业务处理…

GWT主要提供了四种创建GWT项目的命令行工具。分别是：

Creator
projectCreator
applicationCreator
i18nCreator
junitCreator

projectCreator

  • 支持Eclipse IDE
  • 会生成·.class和.project文件，生成后直接导入到Eclipse项目。
  • 支持的参数:
Arguments Description
-eclipse 生成支持Eclipse的文档项目结构
-ant 生成支持Ant的Ant构建脚本
-out 指定项目生成的路径
-overwrite 在编译过程中覆盖原有的文件
-ignore 在编译过程中忽略原有的文件和路径，不进行覆盖

Demo

applicationCreator

  • 用于生成应用程序功能的项目结构，可以通过launch脚本启动项目。
  • 会创建Java的项目结构，包含HTML页面以及Module文件。
  • 会创建命令行脚本文件，即.launch文件。
  • 参数列表与projectCreator一致。

i18nCreator

  • 支持语言国际化。
  • 会创建Java的项目结构，包含HTML页面以及Module文件。
  • 会创建基于键值对的语言配置文件。
  • 参数列表与projectCreator一致。

junitCreator

  • 集成Junit测试框架。
